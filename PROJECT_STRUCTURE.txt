HOW TO USE:
===========

Step 1: Prepare Files
- Place all .h and .cpp files in the same directory
- Place Makefile in the same directory
- Place students_data.csv in the same directory
- Place README.md in the same directory

Step 2: Compile
Command: make
or: g++ -Wall -Wextra -std=c++17 -O2 -g -pthread -o erp_system main.cpp

Step 3: Run
Command: ./erp_system
or: make run

Step 4: Use Application
- Choose login type (Admin or Student)
- Follow menu options
- Perform operations as per role
